# Notepad++ ShortCut Sheet

## BASIC EDIT

| Function | Command | Type |
| --------- | --------- | --------- |
| 上書き保存 | Ctrl + S | Default |
| 行複製 | Ctrl + D | Default |
| 行削除 | Ctrl + X | Default |
| 行先頭へ移動 | Home | Default |
| 行末尾へ移動 | End | Default |
| インクリメンタルサーチ | Ctrl + Alt + Shift + I  | Default |
| 次の検索へ移動 | F3 | Default |
